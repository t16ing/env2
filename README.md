env2: The collections for Linux Shell and Desktop Environment Repository
===

## Why env2?

A dotfiles for tmux, zsh and vim.

Also provide install scripts, and bin files collection for linux shell and desktop.

## Dependency

oh-my-zsh
powerlevel10k
oh-my-tmux
